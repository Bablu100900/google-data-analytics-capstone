# Google Data Analytics Capstone Project: Cyclistic Bike-Share Analysis

## 🚴 Project Overview
Analyze ride data for Cyclistic, a fictional bike-share company, to identify how to convert casual riders into annual members.

## 📊 Tools Used
- R (tidyverse, lubridate, ggplot2)
- SQL (BigQuery)
- Tableau
- Excel

## 🔍 Key Findings
- Casual riders use bikes more on weekends.
- Members ride more on weekdays, likely for commuting.

## 🎯 Recommendations
- Offer weekday promotions for casual riders.
- Introduce loyalty programs.
- Targeted email campaigns based on user type.
